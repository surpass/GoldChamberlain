16.1. PLT.16.1 Support for Markup Head Elements

Portals should indicate if they support the MimeResponse property MimeResponse.MARKUP_HEAD_ELEMENT (value: "javax.portlet.markup.head.element" ) by providing the PortalContext.HTML_HEAD_ELEMENT_SUPPORT (value: "javax.portlet.markup.head.element.support") property on the PortalContext. A non-null value of MARKUP_HEAD_ELEMENT_SUPPORT indicates that the portal application supports the MARKUP_HEAD_ELEMENT property.
